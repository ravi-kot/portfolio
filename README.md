## Deployment

Changes pushed to `main` appear live within 5-10 minutes (GitHub Pages build + CDN cache refresh).

### Hard refresh if changes don't appear:
- **Chrome/Edge:** `Ctrl + Shift + R`
- **Firefox:** `Ctrl + F5`
- **Safari:** `Cmd + Option + R`

## Design Principles

1. **Problem → Solution → Result** — Every project explains the thinking, not just the tools
2. **No buzzword soup** — Technical terms are explained or avoided
3. **Recruiter + Engineer friendly** — Plain English summaries with detailed dropdowns
4. **Production-focused** — Emphasis on real systems, not just notebooks
